# Comandos Bot

Segundo bot do projeto, focado em comandos utilitários.

## Comandos disponíveis
- `/start` - Inicia o bot
- `/help` - Mostra lista de comandos
- `/echo <texto>` - Repete o texto enviado

## Configuração
Use as mesmas variáveis do `.env` principal, ou adicione `BOT_TOKEN_COMANDOS` para um token diferente.
